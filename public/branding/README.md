# Ghost Mascot Logo Options

These SVGs are intended as mascot/logo candidates for Ghost Replay.

- `ghost-logo-option-1-buddy.svg`: round-eyed, extra cute baseline mascot.
- `ghost-logo-option-2-smart.svg`: round glasses, "smart + playful" direction.
- `ghost-logo-option-3-wink.svg`: playful wink expression with more personality.
- `ghost-logo-option-4-scholar.svg`: square glasses for a slightly sharper, thoughtful vibe.

All files use simple vector geometry and a lilac palette so they stay readable at icon sizes.
